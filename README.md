## 🚀 Sobre o Projeto

Este projeto foi desenvolvido como parte de um desafio da plataforma **DIO (Digital Innovation One)** com o objetivo de explorar o uso da **Inteligência Artificial como ferramenta de aprendizado ativo**.

Utilizando o **NotebookLM**, foi criado um **Caderno Temático de Estudos**, reunindo fontes confiáveis, engenharia de prompts e organização de conhecimento para gerar um **miniguia estruturado sobre Inteligência Artificial**.

A proposta do projeto é demonstrar como profissionais de tecnologia podem utilizar IA para **acelerar aprendizado, organizar conhecimento e gerar material de estudo de alta qualidade**.

---

# 🎯 Objetivos do Projeto

* Explorar o uso do **NotebookLM** como ferramenta de estudo
* Aplicar **engenharia de prompts**
* Organizar conhecimento de forma estruturada
* Criar um **miniguia de estudo com IA**
* Demonstrar **pensamento crítico e curadoria de fontes**

# ⚠️ Troubleshooting (Dificuldades Encontradas)

Durante os testes com IA foram observados alguns pontos importantes:

* Prompts muito genéricos geram respostas superficiais
* Prompts específicos produzem respostas muito melhores
* Informar contexto ajuda a IA a responder com mais precisão

### Exemplo

❌ Prompt fraco

```
Explique IA
```

✅ Prompt melhorado

```
Explique Inteligência Artificial de forma simples para iniciantes em programação, incluindo exemplos práticos.
```

# 📖 Miniguia de Estudo

## 📌 O que é Inteligência Artificial?

A Inteligência Artificial (IA) é um campo da ciência da computação focado no desenvolvimento de sistemas capazes de executar tarefas que normalmente exigiriam inteligência humana.

Essas tarefas incluem:

* reconhecimento de padrões
* aprendizado com dados
* tomada de decisões
* processamento de linguagem natural

---

## 💡 Aplicações da Inteligência Artificial

A IA já está presente em diversas tecnologias modernas:

* Assistentes virtuais (Alexa, Google Assistant)
* Sistemas de recomendação (Netflix, Spotify)
* Chatbots
* Diagnóstico médico
* Veículos autônomos
* Ferramentas de programação assistida

---

# 📚 Glossário

**Inteligência Artificial (IA)**
Tecnologia que permite que máquinas executem tarefas que normalmente exigem inteligência humana.

**Machine Learning**
Subárea da IA que permite que sistemas aprendam a partir de dados.

**Algoritmo**
Conjunto de instruções usadas para resolver um problema.

**Modelo de IA**
Sistema treinado para reconhecer padrões e gerar previsões.
